# 📌 Problem Statement

## **Track 3 — Sakhi path: Education, FinTech & Career Empowerment**

Build innovative solutions that improve access to:

- 🎓 Education
- 💰 Financial Literacy & Inclusion
- 👩‍🏫 Mentorship
- 📚 Skill Development
- 💼 Career Growth Opportunities

with a special focus on **women**, especially those aspiring to build careers in **STEM**.

---

# 💡 Proposed Solution

## **Sakhi path**

Sakhi path is an intelligent AI-powered platform designed to bridge the gap between women and career opportunities.

Our solution leverages Artificial Intelligence to provide:

- Personalized career guidance
- AI-powered mentorship assistance
- Skill gap analysis
- Learning roadmap generation
- Resume improvement
- Interview preparation
- Scholarship & opportunity recommendations
- Community support

The platform acts as a digital mentor available 24×7, helping women confidently navigate education, career planning, and professional growth.

---

# 🚀 Inspiration

Millions of talented women lose opportunities because they lack:

- Proper guidance
- Mentorship
- Career awareness
- Personalized learning paths
- Confidence during interviews
- Access to relevant opportunities

We wanted to create a platform that serves as an AI mentor capable of empowering every woman to achieve her career aspirations.

# 🛠️ Installation

Clone the repository

```bash
git clone https://github.com/Sumit-00621/VibeHackers-v2v.git
```

Navigate into the project

```bash
cd VibeHackers-v2v
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```
